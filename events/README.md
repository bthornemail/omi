# OMI Event Models

Events are canonical declarations of observed or requested relations. They are
appendable log records, not callbacks and not imperative mutations.

Required event IDs:

- `event.vfs-resolve`
- `event.lazy-project`
- `event.render-request`
- `event.carrier-scan`
- `event.model-hotplug`
- `event.pointer-select`
- `event.query-request`
- `event.texture-request`
- `event.diagram-request`
