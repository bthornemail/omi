# Phase 63 - OMI Geometric Reconciliation Layer

Phase 63 solidifies the chart/geometry doctrine as its own OMI layer. Charts
are bounded state-spaces, geometry is the lawful edge relation between them,
and reconciliation is the rule that keeps lazy, greedy, static, and animated
views consistent with the same declared object.

## Root Statement

Charts define admissible states.

Geometry defines admissible transitions.

Reconciliation proves that unfolding and refolding preserve identity.

## Four Views

- `lazy` = barcode / sealed address
- `greedy` = chart / unfolded geometry
- `static` = reconciliation of lazy and greedy in declared space
- `animated` = sexagesimal rolling difference across frames, coordinated by the
  `5040` reconciliation court

These are projection modes, not new authority paths.

## Fractal Resolution

Vertex, edge, face, and cell are not separate kinds of thing. They are depth
levels of the same declared state.

- `vertex` = sealed address
- `edge` = lawful transition
- `face` = composed boundary
- `cell` = enclosed state volume
- `subchart` = recursive chart interior

A node may unfold into a subchart, and a subchart may refold into a node,
without changing identity.

- `60` local sexagesimal sweep
