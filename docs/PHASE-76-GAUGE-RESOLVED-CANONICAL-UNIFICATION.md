## Adapter Surface

The new triangulation adapter reads both:

- canonical `.omilisp` declarations
- legacy FS/GS/RS/US trace documents

It emits deterministic S-P-O-M triplets with:

- `subject`
- `predicate`
- `object`
- `modality`

The modality carries the structural planes and gauge hearing metadata so the
same declaration can be triangulated without collapsing identity into view.

- the same canonical declaration yields stable identity receipts
- legacy trace documents and canonical declarations normalize into one
  triangulation surface
- projection/view receipts change with lens mode
- declaration identity does not change when the lens changes
