# OMI Constitutional Geometry

## 0. Preamble

OMI is boundary-first, not parser-first.

Structure exists before projection. Meaning emerges through admissible
traversal over declared segmented structure.

```text
canonical bytes
-> admissible segmentation
-> declared structure
-> gauge law
-> propagation
-> projection
-> receipts
-> presentation
```

This document is the sovereign doctrine source for OMI's constitutional
geometry. Phase notes, runtimes, renderers, workbench systems, packages,
narratives, and transport surfaces derive from this spine.

## 1. Axioms

### Boundary Before Symbol

Declared separation exists before interpretation.

OMI does not begin with:

```text
symbol -> parser -> syntax tree -> semantics
```

It begins with:

```text
boundary -> segmentation -> traversal -> projection
```

### Structure Before Projection

Structure exists before observation.

Projection reveals structure but does not create it. Parsers, renderers,
transports, views, trust records, and narratives are projection surfaces over
declared structure.

### Segmentation Is Topology

Whitespace, spacing, enclosure, quiet zones, packet frames, stream scopes, and
chunk boundaries are structural geometry.

### Traversal Is Meaning

Meaning emerges from admissible traversal through declared structure. Endian
orientation, car/cdr direction, lazy/greedy views, overlays, narratives, and
stream projections are traversal regimes.

## 2. Constitutional Laws

### Omicron Boundary Law

`0x20` is the admissible threshold between latent structure and observable
projection.

```text
0x00..0x1F = latent structural substrate
0x20       = omicron boundary / admissible segmentation
0x21..0x7E = observable projection field
```

Space is not absence. Space is the first printable declaration of separation.

### CONS Aperture Law

CONS is an oriented propagation aperture.

```text
car = opening boundary
cdr = closing boundary
```

A CONS cell is a boundary pair, traversal window, masking aperture, and
projection interval. Endian reversal, directionality, and enclosure inversion
are orientation transforms over that aperture.

### Projection Law

Parsing is projection, not authority.

Every parser, renderer, view, package, carrier, barcode, chart, and narrative
is a lens over declared structure. No projection may rewrite declaration
authority.

### Receipt Separation Law

Receipts preserve different invariants and must not collapse:

```text
identity_receipt   = declaration authority
projection_receipt = readout / reconciliation witness
package_receipt    = transport serialization witness
view_receipt       = presentation state
```

### Propagation Law

```text
Gauge is the local field law.
Wave is propagation through the field.
Receipt is the coherence witness.
Projection is the observed spectrum.
```

Propagation describes behavior through declared structure. It is not a new
runtime authority.

## 3. Derived Geometries

These geometries are consequences of the axioms and laws. They are not
constitutional primitives.

### Pascal / Simplex Geometry

The outer `1`s of Pascal-like rows can be read as CONS enclosure references.
Interior coefficients are propagation multiplicities. Diagonals are
simplex-growth paths through recursive boundary structure.

### Tetrahedral Diagonals

Recursive masking and enclosure produce layered simplex paths. Tetrahedral
diagonals are higher-order propagation routes through nested apertures.

### Gnomon Geometry

Growth occurs by boundary extension. A new layer wraps the old structure
instead of replacing it.

### Inverse-Square Projection

Observation density attenuates with traversal distance and resolution.
Lazy/greedy views, barcode/chart duality, sparse carriers, and distant
projection surfaces are consequences of aperture-limited observation.

### Golomb / ECC Sparsity

Sparse coding records distance, absence, correction, and coherence. Golomb-like
and ECC-like structures are projection witnesses over sparse propagation
fields, not source authority.

### Barcode / Chart Duality

Barcode is compressed carrier observation. Chart is expanded geometric
observation. Both are spectral views over the same declared structure.

## 4. Structural Planes

FS/GS/RS/US are canonical segmentation planes, not metaphysical truths.

```text
FS = full world / document / artifact scope
GS = grouped region / subsystem
RS = record / path / relation
US = unit / token / glyph / cell
```

They give declared structure an addressable topology. They do not replace the
constitutional axioms.

## 5. Semantic Projection

S-P-O-M is derived semantic triangulation, not source authority.

```text
S = source / subject / emitter
P = propagation relation / predicate / edge
O = target / object / resonance point
M = modality / phase condition / authority state
```

Triplets are useful because they expose relation geometry without claiming that
semantic projection is the canonical source.
