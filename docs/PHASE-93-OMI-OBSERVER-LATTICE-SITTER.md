# Phase 93 - OMI Observer / Lattice-Sitter Doctrine

## Summary

Phase 93 defines the OMI observer as a receipt-seeded lattice-sitter: a lawful
observer, translator, and incidence resolver over declared OMI surfaces.

The observer is not an AI agent first and is not an authority. It is a
projection resolver that selects an admissible orientation frame and translates
boundary structure into receipt-verifiable projections.

```text
surface
-> boundary
-> CONS edge
-> binary quadratic relation
-> Fano 7! orientation

-> S-P-O-M declaration
-> receipt
```

The observer does not guess meaning. It uses canonical input and receipts to
select a deterministic orientation frame.

## Fano Orientation Resolver

```text
7 Fano points
-> 7! = 5040 selector permutations
-> receipt-seeded orientation selection
```

The Fano lottery is deterministic, not stochastic. A canonical input produces
an identity receipt, that receipt seeds the selector, and the selected
orientation supplies the local incidence frame.
